\# AI CRM Assistant



AI-powered CRM system that converts doctor interaction notes into structured insights.



\## Features

\- Extract structured data

\- Summarize interactions

\- Suggest next actions

\- Generate insights



\## Tech Stack

\- FastAPI

\- LangGraph

\- React



\## Run



Backend:

cd backend

venv\\Scripts\\activate

uvicorn graph:app --reload



Frontend:

cd frontend

npm install

npm run dev
